## 安装

使用 `npm` 进行安装

```sh
npm install --save a-edge-extends-types@latest
```

## 使用

需在项目的根目录添加 `chrome.d.ts` 文件，文件内容：

```ts
import { Chrome } from 'a-edge-extends-types';

declare global {
  interface Window {
    chrome: Chrome;
  }

  const chrome: Chrome;
}
```

包会在安装或是更新时自动写入该文件，若没有。则需要使用 `npx immm` 创建该文件，并保证 `tsconfig.json` 能正确的识别到该文件即可

当前仅给出了特别少的一部分，如下列表：
